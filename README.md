# Система управления местами в гостинице

Написана полностью с нуля с некоторыми выразками из других проектов

composer install

need nodejs 14

npm i

npm run build

## Без докера
php -S localhost:8000

## Через докер
docker-compose -f docker-compose.yml up --build
